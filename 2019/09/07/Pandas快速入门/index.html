1. 前言

  Pandas是Python的核心数据分析库，一个强大的分析结构化数据的工具集；它的使用基础是Numpy（提供高性能的矩阵运算）；用于数据挖掘和数据分析，同时也提供数据清洗功能。Pandas有Series（1维）和DataFrame（2维）两种主要的数据结构。DataFrame是一个表格型的数据结构，包含有一组有序的列，每列可以是不同的值类型(数值、字符串、布尔型等)，DataFrame即有行索引也有列索引。Series是一种类似于一维数组的对象，是由一组数据(各种NumPy数据类型)以及一组与之相关的数据标签(即索引)组成。

参数 说明

2.4.4 合并

  • merge

参数说明：

参数 说明
left / right 两个DataFrame对象。
how 合并方式：’left’, ‘right’, ‘outer’, ‘inner’，默认为’inner’。
on 指定要联结的列，left和right两个DataFrame有相同column情况下使用。
left_on / right_on 两个DataFrame没有相同情况下，指明左边和右边的参考列。
left_index / right_index 指定是否以行标签索引为参考进行合并。
sort 合并结果是否按on指定的参考column进行排序。
suffixes 合并后如果有重复column，分别加上后缀。
indicator 在合并的DataFrame中新增名为_merge的一列，指明每一行合并键源自哪个DataFrame。
validate 判断合并键中是否存在重复项。
  • concat

常用参数说明：

参数 说明
objs 要连接的对象，如Series、DataFrame。
axis 沿着哪个轴向连接，axis = {0, 1}，默认值为0，沿着行的方向；axis=1，沿着列的方向。
join 处理非合并方向的行（列），join = {‘outer’, ‘inner’}，默认值为’outer’。
ignore_index 默认值为False，合并后是否重建索引。
keys 创建层次化索引，默认值为None。

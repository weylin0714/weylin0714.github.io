Python数据分析常用库（二）--Matplotlib

Matplotlib

Matplotlib是一个非常强大的 Python 画图工具，可以帮助我们以直观的形式更好地了解数据。它支持两种绘图方式，即 ax.plot()和plt.plot()。plt.plot() 适合快速绘图，而 ax.plot() 适合用于精细绘图。下面依次来介绍如何使用。

1. ax.plot() 绘图

首先，我们来理解一些概念：Figure、Axes、Axis。

Figure：如图中所示，可理解为画布，我们所有的内容都会绘制在这个上面。

Axes：可理解为实际绘图的区域，在一个Figure中可以有多个 Axes。

Axis：表示绘图时的横纵坐标轴。

1.1 基本介绍

首先导入 matplotlib 中的 pyplot 模块：

import matplotlib.pyplot as plt

创建一个没有 Axes 的 figure：

fig = plt.figure()

创建 Axes：

fig, ax = plt.subplots()  # 返回一个fig图像对象和ax区块对象
# 等价于
# fig = plt.figure()
# ax = fig.add_subplot(1,1,1)

fig, ax=plt.subplots(m,n,figsize=(a,b))：创建 $m\times n$ 个 Axes 对象，每个大小为 $a\times b$。当创建多个 Axes对象时，返回的 ax 类型为numpy.ndarray。

之后可以通过 ax 调用一系列函数进行绘图操作。

1.2 绘图操作

1.2.1 折线图

ax.plot(x, y, [fmt], [x2], [y2], [fmt2], ..., **kwargs)

fmt = '[marker][line][color]'：可选的，由线条颜色、线型、标记符号组成。

**kwargs：可选的，设置其他属性，如线宽（linewidth）、图例标签（label）、标记符号大小（markersize）等。

详细使用说明查看官方文档。

示例：

import numpy as np
import matplotlib.pyplot as plt

# Fixing random state for reproducibility
np.random.seed(666)

dt = 0.1
t = np.arange(0, 10, dt)
nse1 = np.random.randn(len(t))                 # white noise 1
nse2 = np.random.randn(len(t))                 # white noise 2

# Two signals with a coherent part at 10Hz and a random part
s1 = np.sin(2 * np.pi * 10 * t) + nse1
s2 = np.sin(2 * np.pi * 10 * t) + nse2

fig, ax = plt.subplots()
ax.plot(t, s1, t, s2)
ax.set_xlim(0, 2)   # 设置显示范围
ax.set_xlabel('time') # 设置x轴标签
ax.set_ylabel('s1 and s2') # 设置y轴标签
ax.grid(True)
plt.show()

1.2.2 散点图

ax.scatter(x, y, s=None, c=None, alpha=None, marker=None, ..., **kwargs)

s, c, alpha分别对应散点大小(size)、颜色(color)、透明度。marker：设置点的形状。详细说明查看官方文档。

示例：

import matplotlib.pyplot as plt

plt.rcParams["font.sans-serif"] = "SimHei" # 设置中文字体

# Fixing random state for reproducibility
np.random.seed(19680801)

N = 50
x = np.random.rand(N)
y = np.random.rand(N)
colors = np.random.rand(N)
area = (30 * np.random.rand(N))**2

fig, ax = plt.subplots()
ax.scatter(x, y, s=area, c=colors, alpha=0.5)
ax.set_title('散点图')
plt.show()

设置中文字体：由于 Matplotlib 的默认字体是英文，因此在向图表中添加中文的时候，常常无法显示，通过设置plt.rcParams[“font.sans-serif”] = “SimHei”，可以支持中文字体。

其他常用中文字体：

1.2.3 柱状图

ax.bar(x, height, width=0.8, bottom=None, align='center', **kwargs)

x：柱状图x轴的坐标。

height：各柱子的高度。

width：柱的宽度，默认是0.8，可传入数组，绘制图不等宽的柱状图。

bottom：各柱子底部的y坐标值，默认为0。

align：柱子与x坐标的对齐方式，可设置为 {‘center’, ‘edge’}，默认为’center’。

**kwargs：设置其他属性，如柱子颜色（color）、柱边框颜色（edgecolor）、边框线宽（linewidth）。

详细说明查看官方网站。

示例一：分组柱状图

import matplotlib
import matplotlib.pyplot as plt
import numpy as np

labels = ['G1', 'G2', 'G3', 'G4', 'G5']
men_means = [20, 34, 30, 35, 27]
women_means = [25, 32, 34, 20, 25]

x = np.arange(len(labels))  # the label locations
width = 0.35  # the width of the bars

fig, ax = plt.subplots()
rects1 = ax.bar(x - width/2, men_means, width, label='Men')
rects2 = ax.bar(x + width/2, women_means, width, label='Women')

# Add some text for labels, title and custom x-axis tick labels, etc.
ax.set_ylabel('Scores')
ax.set_title('Scores by group and gender')
ax.set_xticks(x)
ax.set_xticklabels(labels)
ax.set_ylim([0, 40])
ax.legend()

def autolabel(rects):
    """Attach a text label above each bar in *rects*, displaying its height."""
    for rect in rects:
        height = rect.get_height()
        ax.annotate('{}'.format(height),
                    xy=(rect.get_x() + rect.get_width() / 2, height),
                    xytext=(0, 1),  # 1 points vertical offset
                    textcoords="offset points",
                    ha='center', va='bottom')

autolabel(rects1)
autolabel(rects2)
fig.tight_layout()
plt.show()

使用 annotate() 在指定位置添加文本，详情查看官方网站。

示例二：堆叠柱状图

import matplotlib.pyplot as plt

labels = ['G1', 'G2', 'G3', 'G4', 'G5']
men_means = [20, 35, 30, 35, 27]
women_means = [25, 32, 34, 20, 25]
men_std = [2, 3, 4, 1, 2]
women_std = [3, 5, 2, 3, 3]
width = 0.35       # the width of the bars: can also be len(x) sequence

fig, ax = plt.subplots()

ax.bar(labels, men_means, width, yerr=men_std, label='Men')
ax.bar(labels, women_means, width, yerr=women_std, bottom=men_means, label='Women')

ax.set_ylabel('Scores')
ax.set_title('Scores by group and gender')
ax.legend()

plt.show()

1.3.4 水平条形图

ax.barh(y, width, height=0.8, left=None, align='center', **kwargs)

y：y轴每个柱的位置，对应 bar的x。

width：柱的宽度，对应bar的height。

height：柱的高度，对应bar的width，默认值为0.8。

left：条形图左边开始的x轴坐标，对应bar的bottom。

align：柱子与y坐标的对齐方式，可设置为 {‘center’, ‘edge’}，默认为’center’。

详细说明查看官方网站。

import matplotlib.pyplot as plt
import numpy as np

# Fixing random state for reproducibility
np.random.seed(666)

plt.rcdefaults()
fig, ax = plt.subplots()

# Example data
people = ('Tom', 'Dick', 'Harry', 'Slim', 'Jim')
y_pos = np.arange(len(people))
performance = 3 + 10 * np.random.rand(len(people))
error = np.random.rand(len(people))

ax.barh(y_pos, performance, xerr=error, align='center', color='c')
ax.set_yticks(y_pos)
ax.set_yticklabels(people)
ax.invert_yaxis()  # 改变标签的顺序，从上到下
ax.set_xlabel('Performance')
ax.set_title('How fast do you want to go today?')

plt.show()

1.3.5 直方图

ax.hist(x, bins=None, range=None, density=False, histtype='bar', color=None, label=None,...)

x：需要计算并绘制直方图的数组。

bins：控制直方图分箱的个数，默认为10。

range：bins的上下范围，如果未提供，则范围为 (x.min(), x.max())。

density：绘制并返回一个概率密度，默认为False。

histtype：绘制的直方图类型，可设置为{‘bar’, ‘barstacked’, ‘step’, ‘stepfilled’}其中之一, 默认为’bar’。

color：颜色。

label：标签。

详细说明查看官方网站。

示例一：一维数组的直方图

import matplotlib
import numpy as np
import matplotlib.pyplot as plt

np.random.seed(666)

# example data
mu = 100  # mean of distribution
sigma = 15  # standard deviation of distribution
x = mu + sigma * np.random.randn(437)

num_bins = 50

fig, ax = plt.subplots()

# the histogram of the data
n, bins, patches = ax.hist(x, num_bins, density=True)

# add a 'best fit' line
y = ((1 / (np.sqrt(2 * np.pi) * sigma)) *
     np.exp(-0.5 * (1 / sigma * (bins - mu))**2))
ax.plot(bins, y, '--')
ax.set_xlabel('Smarts')
ax.set_ylabel('Probability density')
ax.set_title(r'Histogram of IQ: $\mu=100$, $\sigma=15$')

# Tweak spacing to prevent clipping of ylabel
fig.tight_layout()
plt.show()

示例二：多个数据集的直方图

import numpy as np
import matplotlib.pyplot as plt

np.random.seed(666)

n_bins = 10
x = np.random.randn(1000, 3)

fig, ((ax0, ax1), (ax2, ax3)) = plt.subplots(nrows=2, ncols=2)

colors = ['red', 'tan', 'lime']
ax0.hist(x, n_bins, density=True, histtype='bar', color=colors, label=colors)
ax0.legend(prop={'size': 10})
ax0.set_title('bars with legend')

ax1.hist(x, n_bins, density=True, histtype='bar', stacked=True)
ax1.set_title('stacked bar')

ax2.hist(x, n_bins, histtype='step', stacked=True, fill=False)
ax2.set_title('stack step (unfilled)')

# Make a multiple-histogram of data-sets with different length.
x_multi = [np.random.randn(n) for n in [10000, 5000, 2000]]
ax3.hist(x_multi, n_bins, histtype='bar')
ax3.set_title('different sample sizes')

fig.tight_layout()
plt.show()

1.3.6 饼图

ax.pie(x, explode=None, labels=None, colors=None, autopct=None, labeldistance=1.1, startangle=0, radius=1, counterclock=True,...)

x：数组x。

explode：指定要偏移每个楔体的距离。

labels：图例标签。

colors：每一部分的颜色。

autopct：用数值标记每一部分。

labeldistance：label标记的绘制位置,相对于半径的比例，默认值为1.1, 如<1则绘制在饼图内侧。

startangle：起始绘制角度,默认图是从x轴正方向逆时针画起,如设定为90则从y轴正方向画起。

radius：饼图的半径，默认为1。

counterclock：指定指针方向，默认为True，即逆时针。若设置为False，则为逆时针。

详细说明查看官方网站。

示例一：基本的饼图

import matplotlib.pyplot as plt

# Pie chart, where the slices will be ordered and plotted counter-clockwise:
labels = 'Frogs', 'Hogs', 'Dogs', 'Logs'
sizes = [15, 30, 45, 10]
explode = (0, 0.1, 0, 0)  # only "explode" the 2nd slice (i.e. 'Hogs')

fig1, ax1 = plt.subplots()
ax1.pie(sizes, explode=explode, labels=labels, autopct='%1.1f%%',
        shadow=True, startangle=90)
ax1.axis('equal')  # Equal aspect ratio ensures that pie is drawn as a circle.

plt.show()

示例二：给饼图添加图例

import numpy as np
import matplotlib.pyplot as plt

fig, ax = plt.subplots(figsize=(6, 3), subplot_kw=dict(aspect="equal"))

recipe = ["375 g flour",
          "75 g sugar",
          "250 g butter",
          "300 g berries"]

data = [float(x.split()[0]) for x in recipe]
ingredients = [x.split()[-1] for x in recipe]

def func(pct, allvals):
    absolute = int(pct/100.*np.sum(allvals))
    return "{:.1f}%\n({:d} g)".format(pct, absolute)

wedges, texts, autotexts = ax.pie(data, autopct=lambda pct: func(pct, data),
                                  textprops=dict(color="w"))

ax.legend(wedges, ingredients,
          title="Ingredients",
          loc="center left",
          bbox_to_anchor=(1, 0, 0.5, 1))

plt.setp(autotexts, size=8, weight="bold")

ax.set_title("Matplotlib bakery: A pie")

plt.show()

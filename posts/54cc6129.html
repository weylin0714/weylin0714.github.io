Pandas快速入门

1. 前言

  Pandas是Python的核心数据分析库，一个强大的分析结构化数据的工具集；它的使用基础是Numpy（提供高性能的矩阵运算）；用于数据挖掘和数据分析，同时也提供数据清洗功能。Pandas有Series（1维）和DataFrame（2维）两种主要的数据结构。DataFrame是一个表格型的数据结构，包含有一组有序的列，每列可以是不同的值类型(数值、字符串、布尔型等)，DataFrame即有行索引也有列索引。Series是一种类似于一维数组的对象，是由一组数据(各种NumPy数据类型)以及一组与之相关的数据标签(即索引)组成。

2. Pandas快速入门

2.1 数据结构

2.1.1 Series

  Series是一维索引标记的数组，能够保存任何数据类型（整数、字符串、浮点数、Python对象等）。创建Series的基本方法：

import pandas as pd   # 在使用pandas模块时需要先引入 
s = pd.Series(data, index=index)

其中，data可以是字典、多维数组 、标量等；索引index是轴标签列表。

  • data为多维数组

    索引index的长度要和数据data的长度相同。如果没有传递索引，将默认创建值索引，即[0，1，...，len(data)-1]。

import numpy as np
import pandas as pd
s = pd.Series(np.random.randn(5), index=['a', 'b', 'c', 'd', 'e']) # 指定索引
Out:
    a    0.110595
    b    0.990848
    c    0.382866
    d   -0.536102
    e   -0.062049
    dtype: float64
s = pd.Series(np.random.randn(5)) # 不指定索引
Out:
    0   -0.721240
    1    0.126976
    2    0.465630
    3   -0.375598
    4    0.894133
    dtype: float64
  • data为字典
# 未指定索引，使用Python版本> = 3.6且Pandas版本> = 0.23，则索引将按dict的插入顺序排序，否则将按字典键的词法顺序排序。
d = {'b': 1, 'a': 0, 'c': 2}
s = pd.Series(d)
Out:   # 索引为字典中的键
    b    1
    a    0
    c    2
    dtype: int64
# 指定索引
d = {'a': 0., 'b': 1., 'c': 2.}
s = pd.Series(d, index=['b', 'c', 'd', 'a'])
Out:
    b    1.0
    c    2.0
    d    NaN  
    a    0.0
    dtype: float64

NaN（不是数字）是Pandas中使用的标准缺失数据标记

  • data为标量

    必须提供索引，Series 按索引长度重复该标量值。

s = pd.Series(5., index=['a', 'b', 'c'])
Out:
    a    5.0
    b    5.0
    c    5.0
    dtype: float64

2.1.2 数据帧（DataFrame）

  DataFrame是一个二维标记数据结构，具有可能不同类型的列，可以将其视为电子表格或SQL表。DataFrame可以接受的数据：列表、字典、Series字典、多维数组字典、列表字典等。

  • data为Series构成的字典
d = {'one': pd.Series([1., 2., 3.], index=['a', 'b', 'c']),
     'two': pd.Series([1., 2., 3., 4.], index=['a', 'b', 'c', 'd'])}
df = pd.DataFrame(d)
Out:
       one  two
    a  1.0  1.0
    b  2.0  2.0
    c  3.0  3.0
    d  NaN  4.0
# 指定索引index和列columns
d = {'one': pd.Series([1., 2., 3.], index=['a', 'b', 'c']),
     'two': pd.Series([1., 2., 3., 4.], index=['a', 'b', 'c', 'd'])}
df = pd.DataFrame(d, index=['d', 'b', 'a'], columns=['two', 'three'])
Out:
    two three
 d  4.0   NaN
 b  2.0   NaN
 a  1.0   NaN
  • data为多维数组字典或列表字典
# 不传递索引
d = {'one': [1., 2., 3., 4.], 'two': [4., 3., 2., 1.]}
df = pd.DataFrame(d)
Out:
       one  two
    0  1.0  4.0
    1  2.0  3.0
    2  3.0  2.0
    3  4.0  1.0
# 传递索引
d = {'one': [1., 2., 3., 4.], 'two': [4., 3., 2., 1.]}
df = pd.DataFrame(d, index=['a', 'b', 'c', 'd'])
Out:
       one  two
    a  1.0  4.0
    b  2.0  3.0
    c  3.0  2.0
    d  4.0  1.0
  • data为字典列表
data = [{'a': 1, 'b': 2}, {'a': 5, 'b': 10, 'c': 20}]
df = pd.DataFrame(data, index=['first', 'second'])  # index为行索引
Out:
            a   b     c
    first   1   2   NaN
    second  5  10  20.0
# 指定输出列
data = [{'a': 1, 'b': 2}, {'a': 5, 'b': 10, 'c': 20}]
df = pd.DataFrame(data, index=['first', 'second'], columns=['a', 'b'])
Out:
            a   b
    first   1   2
    second  5  10

2.3 数据选择

  • 将Series看作多维数组–通过索引选择数据

s['b':'c']
Out:
    b    2
    c    3
    dtype: int64
# 隐式整数索引切片
s[0:1]
Out:
    a    1
    dtype: int64
# 掩码
s[s > 1]
Out:
    b    2
    c    3
    dtype: int64
  • Series看作字典–通过键值选择数据
s = pd.Series([1,2,3], index = ['a', 'b', 'c']) 
s['b']
list(s.items())
'b' in s
Out:
    2
    [('a', 1), ('b', 2), ('c', 3)]
    True
  • 使用索引函数–loc、iloc
# loc: 通过显示索引切片或取值
s.loc['a']
Out:
    1
# iloc: 通过隐式索引切片或取值
s.iloc[1]
Out:
    2

2.4 数据处理

df = pd.DataFrame(np.random.randn(4, 4), index=dates, columns=list('ABCD'))
df1 = df.reindex(index=dates[0:4], columns=list(df.columns) + ['E'])
df1.loc[dates[0]:dates[1], 'E'] = 1

2.4.2 排序

Pandas 支持三种排序方式，按索引标签排序，按列里的值排序，按两种方式混合排序。

2.4.2.1 按索引排序
  • Series按索引排序
# Series.sort_index(ascending=True, inplace=False, na_position='last'): 默认ascending=True为升序排序; inplace默认为False，当inplace=True时，修改原数据; na_position默认为'last',将NaN放在最后。
s = pd.Series([1,2,np.nan, 3], index = ['d', 'b', 'c', 'a'])
s.sort_index()
Out:
    a    3.0
    b    2.0
    c    NaN
    d    1.0
    dtype: float64

参数说明：

参数 说明

2.4.4 合并

  • merge
pd.merge(left, right, how='inner', on=None, left_on=None, right_on=None, left_index=False, right_index=False,             sort=True, suffixes=('_x', '_y'), copy=True, indicator=False, validate=None)

参数说明：

参数 说明
left / right 两个DataFrame对象。
how 合并方式：’left’, ‘right’, ‘outer’, ‘inner’，默认为’inner’。
on 指定要联结的列，left和right两个DataFrame有相同column情况下使用。
left_on / right_on 两个DataFrame没有相同情况下，指明左边和右边的参考列。
left_index / right_index 指定是否以行标签索引为参考进行合并。
sort 合并结果是否按on指定的参考column进行排序。
suffixes 合并后如果有重复column，分别加上后缀。
indicator 在合并的DataFrame中新增名为_merge的一列，指明每一行合并键源自哪个DataFrame。
validate 判断合并键中是否存在重复项。 
# 指定一参考列，how='inner'
left = pd.DataFrame({'key': ['K0', 'K1', 'K2', 'K3'],

result = pd.merge(left, right, on='key') # 默认how='inner'
